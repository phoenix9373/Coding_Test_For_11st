# 11번가 코딩테스트

```
  1. 실행방법: index.html을 로드하여 실행합니다.


  2. 모듈 구조
    main.js
      - App.js : 아래 모듈들의 상태를 관리하는 상위 App.
        - StatusBar.js : 시간 및 Back, New 버튼을 나타냅니다.
        - Home.js : 홈 화면을 나타냅니다.
        - AlarmApp.js : 알람 앱.
        - MemoApp.js : 메모 앱.
        - PhotoApp.js : 사진 앱.

  3. 참고사항
    - 홈 화면 Drag & Drop 구현 X
    - 사진 앱 구현 X
```
